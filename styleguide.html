Motion

Animation

Hover a tile to preview the animation. All durations and easings are available as tokens. Use animation-fill-mode: both on entrance animations to avoid flash-of-unstyled content.

Named keyframes

fade-up Standard entrance. Use on cards and sections.
fade-in Simple opacity. Modals, overlays.
scale-in Springy entrance. Badges, dropdowns.
slide-right Sidebar items, list reveals.
pulse-soft Loading skeletons, live indicators.
spin Spinner / loading state.

Stagger utility

Add .stagger to any grid or list parent — child elements automatically receive increasing animation-delay values up to the 6th child (extend in CSS as needed).

Motion & Code

Code blocks

JetBrains Mono at --text-sm. Inline code uses --primary on a tinted background. Block code uses semantic token colors for syntax highlighting.

Inline code

Call loadRepos() on page load and pass the result to renderCard(repo, index) for each item. The cache key is gh_repos_${'{'}GITHUB_USER{'}'}.

Block with syntax tokens

// Fetch public repos with localStorage cache
async function loadRepos() {
  const cached = getCached();
  if (cached) return render(cached);

  const res = await fetch(
    `https://api.github.com/users/${'{'}GITHUB_USER{'}'}/repos`
  );
  const repos = await res.json();
  setCache(repos);
  render(repos);
}
